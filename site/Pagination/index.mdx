# Pagination 分页

采用分页的形式分隔长列表，每次只加载一个页面。

## 何时使用

- 当加载/渲染所有数据将花费很多时间时；

- 可切换页码浏览数据。

## API

| 属性 | 说明 | 类型 | 是否必填 | 默认值 |
| ---- | ---- | ---- | ------ | ------ |
| total | 数据总数 | number | 是 | - |
| pageSize | 每页条数 | number | 否 | 10 |
| current | 当前页数 | number | 否 | - |
| onChange | 页数改变回调 | (page: number) => void | 否 | - |
