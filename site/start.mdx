# 快速上手

RATurbo-components 致力于提供给程序员愉悦的开发体验。

> 在开始之前，推荐先学习 [React](https://reactjs.org/) 和 [ES2015](https://babeljs.io/docs/en/learn)，并正确安装和配置了 Node.js v8.9 或以上。
官方指南假设你已了解关于 HTML、CSS 和 JavaScript 的中级知识，并且已经完全掌握了 React 的正确开发方式。如果你刚开始学习前端或者 Vue，将 UI 框架作为你的第一步可能不是最好的主意。

## 第一个例子
这是一段简单的样例代码

```jsx
import React, { useState } from 'react';
import { render } from 'react-dom';
import { Drawer } from 'raturbo-components';
import 'raturbo-components/lib/drawer/style';

const App = () => {
  const [open, setOpen] = useState(null);

  return (
    <section>
      <Drawer open={open} openChange={_o=>setOpen(_o)}>
        <div>base demo</div>
      </Drawer>
    </section>
  );
};

render(<App />, document.getElementById('root'));
```


## 按需加载
`RATurbo-components` 默认支持基于 ES modules 的 tree shaking，对于 js 部分，直接引入 import { Button } from 'raturbo-components' 就会有按需加载的效果。

```javascript
import { Drawer } from 'raturbo-components';
import 'raturbo-components/lib/drawer/style';
```

## 使用babel-plugin-component

```bash
yarn add babel-plugin-component --dev
```

如果你使用了 babel，那么可以使用 [babel-plugin-import](https://www.npmjs.com/package/babel-plugin-import) 来进行按需加载，加入这个插件后。你可以仍然这么写：

```javascript
import { Drawer } from 'raturbo-components';
```

修改.babelrc.js
```javascript
{
  "plugins": [
    [
      "import",
      {
        "libraryName": "raturbo-components",
        "libraryDirectory": "lib",
        "style": true // or 'css'
      },
      "raturbo-components"
    ]
  ]
}
```
